python3

低版本py3可选择使用wheel方式或exe方式进行安装。
在该网址下载对应版本的.whl或exe文件。

http://www.voidspace.org.uk/python/pycrypto-2.6.1/

wheel方式

将其放在py3安装目录下的scripts文件夹中。
安装姿势：
打开cmd

exe方式

双击运行，和普通软件安装程序类似

python2

以上python3的方法python2可通用。

python2可以pip直接安装pycrypto

关于pycryptodome

这是另外的大佬替代多年未更新的pycrypto而写的更新版本。网上很多大佬都说它可以直接在python3上面直接pip安装运行，但我试了一天后未果，python2的pycrypto因为年代问题一些新的模块并没有。愁眉莫展之际想到将其装在python2上如何？发现完美补充了新功能并且没问题。
